# 文科生的“出路”
## 来自县城的本科生
我来自一座经济不发达的某县城，像不少小镇做题家一样，见识不多，但是心气很高，希望未来能够有所成就。大一的时候我压着分数线进入了哲学专业学习，同家里人沟通后，大二转到了经济学，动机当然是为了赚钱。尽管我对金钱的态度暧昧不明，但是不可置否的是，这是我的核心动机。显然，哲学专业并不有利于走向这条道路。
在 2018 年的中国，金融行业从业人员似乎还是一个很体面的工作，似乎在一所普通的 211 大学毕业后就能进入金融行业，然后便有机会在投行中叱咤风云，过上体面的生活。
我尚且算是努力的学习经济学，加之我运气很好，成功拿到了学校的保研名额，成为了经济学专业的硕士生。
我其实并不算是一个未雨绸缪的人，作为一个 intp ，似乎不到最后关头，我都是浑浑噩噩的度过，按照普通好学生的路线学习，拿到高的绩点，然后保研，业余时间基本上都沉浸在游戏之中。学校也有一些所谓就业指导课程，但懂得都懂。我并不喜欢这样的生活，我没有找到我人生的”出路“。
此时，我回想起我的本科生生活，我似乎一直活在一种焦虑之中，我懒惰，没有动力去过上我真正想过的生活，我常常感受到我此刻的无意义，我的意思是，我隐约感受到，世界的向我拉下了帷幕，我看不到帷幕后面的混乱与秩序。我在大学的这个剧场中按照某些被安排好的剧情看着“我”过完了我的四年，我越是得要依靠保研为我争取更多的学生时代的生活，给我更多的时间尽可能看见帷幕后的真相。
大四的时候，是我的空白时期，这段时间我没有找到自己的“出路“，一些更加熟悉社会规则的人，同样是经济学保研的同学，已经去找实习了。大四的时候，我们经历了疫情，显然实习并不好找，我算是度过了比较丰富的大四生活，有很多的社会活动，才开始有时间了解一个城市的风土人情。

## 谢幕后的再演
我浑浑噩噩的演出然后谢幕，但我还是没有看见帷幕后的生活。我被安排到了另外一个剧场
此时我开始感受到了大家在考虑之后应该找什么样的工作，我们开始感受到了前一幕戏剧与下一幕戏剧之间的脱节。
经济学专业太难进入投行工作了，对于我不算优秀的本科和研究生学历，我很难进入投行工作，你需要有很丰富的实习经历，而且你应该长袖善舞，显然这不是一个懒惰的 intp 会去干的事情，我也尝试过去面一些投行，但显然效果不好。在一个中部城市读研，你也很难找到投行的实习机会。
当然最好找工作的是你应该给投行投资，成为关系户。不然实在太难了。 2018 年和 2022 年，差距太大了。金融行业急转直下，工作一年比一年难找。
最后我的观察，似乎我的同学基本上都是去了银行或者是考公。我好像没有听说最后去投行工作的。

我感觉到了欺骗，也有一种极大的焦虑，围绕着我的生活。
我其实一直都比较喜欢计算机，首先我比较喜欢玩游戏，其次我比较喜欢捣鼓东西，我本科的时候也浑浑噩噩的学了一点点爬虫，非常少的爬虫知识，基本上就是跑一个 python 程序，上了一些普通的课程，总之约等于没学，实际上也是没学。
但我感觉我对此是感兴趣的，但我一直没有好好的去学习。因为我得保研，保研带来的课业压力确实让我没有心力去学习其他的东西。

所幸，我在研究生生活中，选择一位很好的导师，从来没有给我们安排任何事务性的事情，这几乎是很难想象的，但是我运气确实很好，幸好我保研了，幸好我选中了非常好的导师。当初选择这位老师的很大原因是，我的导师虽然是经济学的博导但本硕博都是哲学专业，然后我又很懒，这位老师不知道为什么没有人选，可能是老师最近都不太发论文吧，然后就最终选择这个老师做我的研究生导师。

突然有一天，我觉得不对，我觉得太不对了，为什么我有种活人被尿逼死的感觉。我感觉我的人生一片灰暗啊，我没有办法进入投行工作，拿到高薪，剩下的工作基本上都是万元以下，而且也要付出很大的代价，而且我也不是长袖善舞的人。
说是我很难接受万元左右的工资，但如果是在纯文科专业，比如中文、历史这种，好像万元已经算是高薪了。如果是考公

而且经济学专业实习也是基本上是倒贴钱，而这个专业通常要求更高的分数。

## 孤注一掷的自救吗，但是 intp
我不觉得这是合理的，回到开始，我似乎挺享受自己坐在电脑面前的生活。所以我不断的问自己，问自己究竟应该去做什么，我感觉到了奇怪，我的四年本科和一年研究生生活究竟意味着什么，我为什么非得要走这一条路，我难道没有别的路可走吗，我学了这个专业我就必须在这个专业的辐射范围内工作吗。
我感觉到太不灵活了，太不人性了，一个人当然会选错专业，因为一道道帷幕遮掩了我们的视线，我们看见的只有一些老旧的、既成的叙事，我们看着自己继续走在这个叙事中，我们不知道自己究竟能怎么做。
既然如此，我们非得继续出演这幕戏剧吗。他没有给我我需要的安全感，没有给我一个好的未来，我不知道谁是编剧，但是我非常讨厌这个编剧，前言不搭后语，形式急转直下。

难道别人不是这样吗。
于是我选择了计算机专业，研一结束的那个暑假，我开始了计算机专业的学习，我觉得我浑浑噩噩的学习了四年，大部分知识都是无用的，难道计算机专业教育就不是这样的吗，真正有用的知识是不是就那几个，于是我开始了以就业为目的，给自己最多一年半的自学时间（赶上研三的秋招甚至春招）。

大部分人在专业学习上难道不是实际上只需要一年的时间学习就足够了吗，大部分的时间都无非是在学课业外的事情，游戏或者别的。

基于此，我决定苦修一年的计算机知识，然后工作。

## 一张更大的帷幕
我究竟应该学习什么知识，我开始在 b 站上看各种视频，一开始是打算成为后端工程师，然后学习java，后面发现自己根本无法进入后端工程师的语境，我按照b站上的网课搭起了后端服务，然后按照好学生应该做的那样做笔记，学习，后面发现这种路子效率太低了，我又一次叩问自己，死磕这些理论知识有什么用呢，我感觉我自己又无所适从的学习了半年左右的计算机知识，学习了 python 学习java，学习算法知识。
但是我不知道学这个究竟有什么用。
我继续问自己，为什么要学这些，所有业界核心的目的不就是要搭建起一个软件服务吗。

于是我开始学习， flutter 此时，我才真正入门了软件工程的世界。当我有了制作一个的目标后，我才会寻找积木，将零碎的知识串成语境。但显然，我当时的技术还是太差了，我的第一个软件制作最终不了了之。

但是幸运的是，利用这个项目我开始参加了一场加密的黑客松，在那里我最终确定我会成为前端开发工程，我用 nextjs 构建了一个项目，并和同行的朋友构建起了一个软件 youbetdao ，当时我的眼光很好，抱上了大佬的大腿，在大佬的帮助下这个项目拿到了很多奖项，尽管后面我中途退出了，后面这个项目在大佬的努力下，又拿到了奇迹创坛的融资。

现在发现我好像把这个项目完全的删除了。。。可惜。。。

理论和工程之间的差距就是是否有一个基于目标的统观的知识，如果没有构建起一个完整的视角，那么将永远不入其门，我也不知道理论的意义。

当时我还在参加哲学专业的读书会，我在想我能不能做一款软件，能够在 pdf 的指定位置，插入音频进行播放，当时我在使用 notability 后面我就想，我能不能自己做一款软件，实际上 notability 是无法在界面中的某个位置插入指定的元素的，于是我开始了没日没夜的研发工作，在继续参与读书会之外，我就是在不断的学习。

我在做这个软件之前，我学习了很多零碎的理论知识， sql， javascript
